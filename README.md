# Flower_Shop
A responsive and user-friendly Flower Shop website built using HTML, CSS and PHP. This project allows customers to browse beautiful flower collections, view product details, and place orders online. It is designed with a clean UI to provide a smooth shopping experience.



# 🌸 Flower Shop Web Application

A modern and responsive Flower Shop web application built using **HTML, CSS, JavaScript, PHP, and MySQL**. This project provides users with an easy and elegant way to browse flowers, view details, and place orders online.

---

## 🚀 Features

- 🌷 Beautiful and responsive UI design  
- 🛒 Product catalog with images and details  
- 🔍 Search and navigation system  
- 🧾 Shopping cart functionality  
- 💳 Order placement system  
- 🔐 User login & registration (if included)  
- 📱 Mobile-friendly design  

---

## 🛠️ Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** XAMPP / Apache
